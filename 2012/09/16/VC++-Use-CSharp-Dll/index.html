    配置属性 -> 常规 -> 项目默认值 -> 公共语言运行时支持 -> 公共语言运行时支持(/clr)

  2. 加入代码
  • 类库路径 #using "./bin/CSharp.dll"

  • 使用类成员和方法

方法一:

void TestFunc()
{
    CSharpClassSample ^cs = gcnew CSharpClassSample();

    cs->member_sample = 1;
    cs->func_sample();
}

方法二:

ref class A{
public:
    static CSharpClassSample obj;
};

void TestFunc_1(void)
{
    A::obj.member_sample = 1;
}

void TestFunc_2(void)
{
    A::obj.func_sample();
}
